### 8.1 즉시 필요 (이메일 로그인이 안 됨)

| 항목 | 어디서 받기 | 환경변수 | 비고 |
|---|---|---|---|
| **Resend SMTP API Key** | https://resend.com/api-keys | (Supabase Dashboard 직접 입력) | `docs/USER_ACTIONS.md` ⑤ 가이드 참조 |
| 도메인 인증 DNS (Resend) | Resend Domains → conatusipsi.com → 안내 받은 MX/TXT 3개 | 가비아 DNS 입력 | SPF + DKIM + MX |

→ 이걸 안 하면 회원가입 인증 메일이 안 가서 가입 자체가 막힘.

### 8.2 결제 활성화 (현재 모든 결제 503)

| 항목 | 어디서 받기 | 환경변수 |
|---|---|---|
| **사업자등록증** | 홈택스 (사업자등록 → 통신판매업 신고) | `NEXT_PUBLIC_BIZ_NAME`, `_REPRESENTATIVE`, `_REGISTRATION_NUMBER`, `_TELECOM_NUMBER` |
| **토스페이먼츠 가맹점 가입** | https://www.tosspayments.com | `TOSS_SECRET_KEY` (secret), `NEXT_PUBLIC_TOSS_CLIENT_KEY` (public) |

→ 사업자등록증 없으면 토스 가맹 신청 불가 → 결제 활성화 불가.

### 8.3 소셜 로그인 (선택)

| 항목 | 어디서 | 환경변수 |
|---|---|---|
| **카카오 OAuth** Client ID/Secret | https://developers.kakao.com → 앱 생성 → REST API 키 + Client Secret | Supabase Console → Authentication → Providers → Kakao 에 직접 입력 (env 불필요) |

→ 카카오 로그인 버튼은 노출되지만 클릭 시 fail.
