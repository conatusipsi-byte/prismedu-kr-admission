## 결정해 주실 사항

사이트의 핵심 데이터인 **한국 대학·학과 정보 + 입학전형 정보**를 어떻게 채울지 두 가지 방안 중 선택해 주십시오. 하이브리드(C)도 가능합니다.

| 옵션 | 방식 | 장점 | 단점 |
|---|---|---|---|
| **A** | 대표님이 직접 데이터 제공 (엑셀/CSV) | 정확도·완성도 보장, 출처 통제권 100% | 입력 부담 큼 (대학 약 200개 × 학과 수천), 매년 직접 갱신 |
| **B** | 공공데이터 API 자동 수집 (**추천**) | 자동 수집, 매년 갱신 스크립트 1회 실행, 무료 | 제공 범위 한계 (등급컷·백분위 미포함 — 아래 설명) |
| **C** | 하이브리드 | 작업 분산 + 품질 확보 | — |

**B 또는 C를 선택하시는 경우** 아래 내용대로 진행됩니다.

| **3** | 대학정보공시 | 대교협 (대학알리미 연계) | 신입생 성적분포(상위 50/70/90% 평균등급), 등록금, 졸업생 취업률, 충원율 | 10,000 |

## 🟥 API로 받을 수 없는 데이터 (사전 공유 필요)

| 데이터 | 어디서도 공개 API 없음 | 대응 방안 |
|---|---|---|
| **합격생 70%컷 / 50%컷 등급·백분위** | 진학사·대학어디가 독점 (사용자 자기제출 데이터 누적) | **본 사이트도 동일 구조로 사용자 누적 필요.** 출시 직후엔 표본 부족 학과는 "데이터 부족" 표시 (CLAUDE.md 정직성 원칙) |
| **올해 경쟁률 실시간** | 원서접수 기간만 대학별 사이트에 표시 | 지난해 경쟁률은 #3 공시 API로 수집 가능 |
| **논술·면접 기출문제** | 대학 자체 공개만 | 출시 후 사용자 제보 누적 |
| **모집요강 본문 (세부 텍스트)** | PDF 링크만 제공, 본문은 직접 파싱 필요 | 보유한 PDF 6개 (고려대·한외대 등) → ETL 스크립트로 파싱 (코드 이미 준비됨) |

## 📋 받는 방법 — 공통 절차 (4개 모두 동일)

### Step 1 · 회원가입 (5분)
- https://www.data.go.kr 접속 → 우측 상단 **회원가입**
- 일반회원(개인) 선택 → 휴대폰 본인인증
- 즉시 로그인 가능

### Step 2 · API 4개 각각 활용신청 (10분)
- 상단 검색창에 키워드 입력 → **오픈API** 탭 → 해당 API 클릭 → **활용신청** 버튼

| API | 검색 키워드 |
|---|---|
| #1 입학전형 | `대학입학전형` |
| #2 대학·학과 마스터 | `대학 학과 정보` 또는 `학교 기본정보` |
| #3 대학공시 | `대학정보공시` 또는 `대학알리미` |
| #4 수능 통계 | `수능 통계` 또는 `대학수학능력시험` |

**신청서 작성 (4개 동일하게)**:
- 활용목적: `한국 대학 입시 정보 안내 웹서비스 운영`
- 시스템유형: **REST**
- 일 트래픽: **10,000건** (10K까지는 자동승인 한도)
- 라이센스: 동의

### Step 3 · 승인 대기 (즉시 ~ 2영업일)
- 대부분 **즉시 자동승인**
- 일부는 검토 후 1~2일
- 알림 메일 또는 마이페이지에서 확인

### Step 4 · 인증키 확인
- 마이페이지 → **개발계정** → 신청한 API 클릭
- **일반 인증키 (Encoding)** + **Decoding** 두 개 표시됨
- 개발팀이 사용할 키는 **Decoding 키** (URL 인코딩 이슈 없음)

전달 시 함께 알려주실 것:
- 각 API의 **엔드포인트 URL** (마이페이지 → 개발계정 → 해당 API 상세화면에 표시됨)
- 응답 **샘플 1~2건** (마이페이지 미리보기 호출 결과 캡처 또는 텍스트)

## 💰 비용

- **공공데이터 API 4개: 전부 무료**
  - 일 한도 10,000건, 한도 초과 시 추가 신청으로 100,000건까지 증액 가능
- **별개로 발생하는 운영 서비스 비용** (참고):
  - Firebase: 무료 → 트래픽 늘면 Blaze 종량제 (월 2~10만원 예상)
  - Toss Payments: 가맹점 수수료만 (결제건당 1.x~2.x%)
  - Anthropic Claude (AI 챗): 호출당 과금, 캐싱 적용으로 절감 중

## ✅ 요약

- 한국 대학 데이터는 정부 공공API 4개로 자동 수집 가능합니다 (대교협·교육부·평가원, 전부 무료).
- 다만 **합격 등급컷·백분위는 어디든 공개 API에 없어** 운영 누적 데이터로 자체 추정해야 합니다 — 진학사·어디가도 같은 방식입니다.
- 출시 직후엔 표본 부족 학과는 "데이터 부족"으로 표시하고, 시즌(7~9월) 데이터가 누적되면서 정확도가 올라가는 구조입니다.

선택 결과 알려주시면 그 방향으로 진행하겠습니다.
